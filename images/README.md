# Images pour FREHEALTH

Ce dossier contient les images utilisées sur le site FREHEALTH.

## Images nécessaires

### 1. `hero-sante-bien-etre.jpg`
**Section : Hero (page d'accueil)**

**Suggestions d'images :**
- Personnes en bonne santé, souriantes, dans un contexte de bien-être
- Sport accessible (marche, yoga, activités douces)
- Diversité des corps et des âges
- Ambiance positive et rassurante
- **À éviter :** Bodybuilders, clichés extrêmes, images trop médicales/froides

**Dimensions recommandées :** 1000x600px (ratio 5:3)
**Style :** Moderne, humain, accessible, chaleureux

### 2. `solution-interface.jpg`
**Section : Solution FREHEALTH**

**Suggestions d'images :**
- Capture d'écran ou mockup d'interface de suivi médical
- Dashboard santé moderne et épuré
- Application mobile ou web de suivi de santé
- Illustration de la plateforme en action
- **Style :** Moderne, épuré, crédible médicalement, professionnel

**Dimensions recommandées :** 800x800px (ratio 1:1)
**Style :** Interface moderne, design épuré, couleurs cohérentes avec la charte (vert/blanc)

## Notes
- Toutes les images doivent être optimisées pour le web (format JPG ou WebP)
- Respecter les droits d'auteur - utiliser des images libres de droits ou créer vos propres visuels
- Sources d'images libres : Unsplash, Pexels, Pixabay (avec filtres "health", "wellness", "medical app")
